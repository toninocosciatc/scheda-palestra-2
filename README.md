# Scheda Palestra PWA — Versione 3

## Modifiche
- I pulsanti `Apri tutti` e `Chiudi tutti` sono stati sostituiti da icone compatte.
- La sezione riassuntiva del giorno è stata ridotta.
- Rimossi `Volume oggi` e `Vs ultima volta` dalla testata.
- La stanchezza ora si imposta con un cursore colorato da 1 a 5.
- Ogni esercizio ha uno `Storico esercizio` richiudibile.
- Lo storico mostra le ultime 3 esecuzioni e, se ce ne sono di più, puoi aprirle tutte all'occorrenza.

## Compatibilità
La chiave di salvataggio resta `scheda-palestra-v1`, quindi i dati precedenti non vengono cancellati.

## Aggiornamento su GitHub
Carica e sostituisci nella root del repository:
- index.html
- style.css
- app.js
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- README.md

Poi clicca `Commit changes`.
